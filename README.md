# MindMate AI

MindMate AI is a comprehensive study and productivity tool powered by AI.

## Project Structure

- **Web App**: Built with React and Vite. Primary interface for web and cross-platform access.
- **Android Native**: Located in the `android-native/` directory. A fully native Android implementation built with Kotlin and Jetpack Compose.

## Native Android Features

- **Modern UI**: Clean Material 3 design with a consistent theme.
- **AI Integration**: High-performance PDF processing, summarization, and quiz generation using Gemini Pro.
- **Local Storage**: Secure history management and session data.
- **Performance**: Native Jetpack Compose UI for a fluid user experience.

## Getting Started (Android)

1. Open the `android-native/` directory in Android Studio.
2. Ensure you have the latest Android SDK and Kotlin version.
3. Build and run on an emulator or physical device.
